# Sign Language Translator

A real-time sign language recognition system using MediaPipe and Machine Learning.

## Features
- Real-time hand gesture detection
- MediaPipe-based landmark extraction
- Trained ML model for sign classification
- Webcam-based live prediction

## Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy
- TensorFlow / Scikit-learn

## How to Run
```bash
pip install -r requirements.txt
python mediapipe_cam_next.py
